DARC theme

A reveal.js port of the DARC zensical theme — Inter + Sligoil Micro, ember on slate.

Headings & body copy

Body text rides on Inter at 36px. Headings drop to Sligoil Micro, the same display face used on the DARC docs site.

Section heading (h3)

Paragraphs pick up strong text, italics, and inline links in the orange ember accent. Selection mirrors the brand color.

Lists

  • Bullets land in the accent color via ::marker
  • Nested lists stay legible
  • Sub-points:
    • square nesting
    • then circles further down
  1. Ordered lists
  2. also keep accent markers

Code

Inline like const slide = darc() renders on a panel chip.

// Block code uses the DARC panel chrome
async function loadDeck(url) {
  const res = await fetch(url);
  return await res.json();
}

Blockquote

A short pull-quote rendered with an ember left rule instead of the default italic shadow box. Legible at slide distance, on-brand at any.
